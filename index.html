# BHash

BHash is a Bangla CSS and/or library that helps you to make your website more beautiful and attractive. It is a very simple and easy to use library. You can use it in your website by just adding a single line of code.

# Quick Start

1. Create a HTML5 index.html file in your root project. Make sure to add viewport. Or you can copy the code mentioned below and paste it in your IDE.

  ./index.html

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Website</title> </head> <body> <h1>Hello World</h1> </body> </html>

2. Add our CDN links in your <head> tag. You can copy the code mentioned below and paste it inside your head tag.

  ./index.html

<!-- BHash CSS --> <link rel="stylesheet" href="https://w3nabil.github.io/bh/instant.css"/> <!-- BHash JS --> <script src="https://w3nabil.github.io/bh/instant.js"></script>

3. Run your file and see the result in your website :))

Also Read : You can also call our library in preload which is recommended for large traffic website.

  ./index.html

<!-- Preload the CSS and JS file --> <link rel="preload" href="https://w3nabil.github.io/bh/instant.css" as="style"/> <link rel="preload" href="https://w3nabil.github.io/bh/instant.js" as="script"/> <!-- Call the CSS and JS file --> <link rel="stylesheet" href="https://w3nabil.github.io/bh/instant.css"/> <script src="https://w3nabil.github.io/bh/instant.js"></script>

4. Read our documentation
